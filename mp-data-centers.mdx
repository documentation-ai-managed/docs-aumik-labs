## Why Madhya Pradesh?

- 25% CAPEX subsidy (max ₹125 Cr for anchor data centers)
- 100% stamp duty reimbursement
- 7-year electricity duty exemption
- Fixed ₹5/unit power for 5 years
- Ranked 4th in Ease of Doing Business

## Government Incentives

### IT Investment Promotion Policy 2023

| Incentive | Benefit |
|-----------|---------|
| Capital Expenditure | 25% on GFCI |
| Women Entrepreneurs | +2% additional |
| Interest Subsidy | 6% on loans, max ₹5 Cr/5 years |
| Stamp Duty | 100% reimbursement |
| Electricity Duty | 100% reimbursement |
| Power Tariff | ₹5/unit fixed for 5 years |

### Infrastructure Subsidy
- Bare shell: ₹2,000/sq.ft
- Furnished: ₹3,000/sq.ft

## Key Contacts

### Application Process
1. Register on invest.mp.gov.in
2. File intention to invest
3. Single window clearance
4. Land allotment (if needed)
5. Apply for incentives

## Cost Estimates

### 50-Rack Facility (500kW IT load)

| Item | Cost |
|------|------|
| Racks (50x42U) | ₹10-25L |
| PDUs | ₹8-15L |
| Cooling | ₹35-60L |
| UPS | ₹30-50L |
| DG Set | ₹30-45L |
| Fire Suppression | ₹15-25L |
| Security | ₹8-15L |
| Network | ₹15-25L |
| **Total CAPEX** | **₹1.5-2.6 Cr** |

**After 25% subsidy:** ₹1.1-2 Cr net

### Monthly OPEX
- Power: ₹25-35L
- Staff (10): ₹5-8L
- Internet: ₹2-5L
- Maintenance: ₹2-3L
- **Total:** ₹35-50L/month

## Quick Start Actions

**Week 1:**
- Register on invest.mp.gov.in
- Contact MPSeDC: 0755-2518585
- Site visit: Indore (Pithampur, Crystal IT Park)

**Week 2:**
- Electrical assessment of your building
- RFQs to rack vendors
- Meet RackBank (local success story)
